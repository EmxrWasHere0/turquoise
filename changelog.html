Initial Public Test

TurquoiseLinux is ready to be tested on real hardware. Here's what it requires:

  • 30GB+ storage
  • 4GB+ RAM
  • x86-64, 4-core CPU
  • UEFI firmware
The test ISO is ~4GB and requires at least 8GB USB storage device to be written.

Used Libraries

  • SysLinux for BIOS system bootloading*
  • GRUB for UEFI system bootloading
  • KDE Plasma for desktop environment
  • KWin for window management
  • Wayland for graphical processing
  • SystemD for service and initialization process

*Only in live media.
